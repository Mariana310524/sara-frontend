# SARA Frontend

Plataforma de análisis y gestión de información corporativa desarrollada con React y Vite.

## Tecnologías utilizadas

* React
* Vite
* React Router
* React Icons
* Recharts
* CSS moderno
* Vercel

## Funcionalidades

* Dashboard ejecutivo con KPIs
* Gráfico de métricas semanales
* Integración Gmail (modo demostración)
* Gestión de archivos
* Historial y errores
* Navegación responsive

## Despliegue

Aplicación publicada en Vercel:
